프로젝트 소개

이 프로젝트에서는 WebSocket을 활용한 실시간 채팅 애플리케이션을 처음부터 끝까지 구현합니다. 단순한 메시지 전송을 넘어, 실무에서 필요한 사용자 인증, 채팅방 관리, 파일 전송, 읽음 표시 등의 기능을 모두 구현하며 실전 감각을 키울 수 있습니다.

Node.js와 Socket.io를 활용한 백엔드 구축부터, React를 사용한 인터랙티브한 UI 개발까지 풀스택 개발의 전 과정을 경험하실 수 있습니다.

학습 내용

  • WebSocket과 Socket.io를 활용한 실시간 통신 구현
  • JWT를 활용한 사용자 인증 시스템 구축
  • React Hooks를 활용한 상태 관리
  • MongoDB를 활용한 채팅 기록 저장
  • 파일 업로드 및 이미지 공유 기능
  • 읽음 표시 및 타이핑 인디케이터 구현
  • 반응형 UI 디자인
  • Docker를 활용한 배포
